1. Демонстрация работы GC

Объект User с двумя полями int и String будет занимать:
new User(1, "a");
Заголовок: 16 байт(64bit jvm)
int: 4 байт
String: 60 байт:
         Заголовок: 16 байт(64bit jvm)
         Поля int(offset, count, hash): 4*3=12 байт
         Ссылочная переменная на объект массива char[]: 8 байта(64bit jvm)

         Массив char[1]:
         Заголовок массива: 16 байт(64bit jvm) + 4 байт на длину массива = 20 байт
         Примитив char("a"): 2 байт
         выравнивание для кратности 8: 2 байт
         итого 24 байт
Итого объект займет 80 байт

Пустой объект без полей займет 16 байт на заголовок.

При ключах -Xmx3m -Xms3m GC вызвался на 5000 объектах
При ключах -Xmx4m -Xms4m GC вызвался уже на 5100 объектах
при -Xmx5m -Xms5m снова на 5000
